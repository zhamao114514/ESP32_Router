# ESP32-S3 校园网共享热点
 ESP32-S3 实现校园网多设备共享上网，使用MAC克隆 + 自动Portal认证。
 ## 功能
 - STA连接校园网，AP开启本地热点
 - MAC伪装绕过单设备登录限制
 - 自动网页认证、掉线重连、心跳保活
 - NAT网络转发，多设备共用网络
 ## 环境要求
 - 硬件：ESP32-S3
 - 编译环境：Arduino IDE，ESP32 核心 ≥ 2.0.14
 - 配置：开启 PSRAM、IPv4+IPv6 Full LWIP、Huge APP 分区
 ## 快速配置
 修改代码顶部 `用户配置区`：
 1. 校园网WiFi名称、密码
 2. 认证服务器IP、端口
 3. 上网账号、密码
 4. 自建热点名称、密码
 5. 已认证设备的真实MAC地址
 ## 使用
 1. 配置参数 → 编译上传
 2. 串口监视器 115200 波特率查看状态
 3. 连接设备热点即可上网
 ## 注意事项
 1. 禁止公开明文账号、密码、MAC地址。
 2. 关闭手机/电脑WiFi随机MAC功能。
 3. 供电不足会导致WiFi频繁断连，请使用稳定电源。
